## 2) 二级页面重塑（首页不变）

建议从首页四个快捷入口和热门服务进入统一二级架构：

1. **美美看脸中心**
   - 页面：检测入口 -> 报告 -> 医生问答 -> 方案 -> 套餐 -> 时间 -> 订单 -> 支付 -> 成功
   - 对应素材：`4-12`

2. **数字人面诊中心**
   - 子流1（医美）：复用美美看脸后半链路（问答/方案/预约）
   - 子流2（中医心理）：机构 -> 医师 -> 问诊 -> 报告 -> 调理 -> 到店预约 -> 复诊提醒 -> 评价
   - 对应素材：`27-36`

3. **Wardrobe 衣柜中心**
   - 页面：衣柜总览 -> 分类管理 -> AI穿搭 -> 收纳建议 -> 衣物管理
   - 对应素材：`13-16`

4. **生活服务中心**
   - 上门服务：`17`
   - 线上瑜伽：`19/20`
   - 线下瑜伽：`18`

5. **电商与比价中心**
   - AI比价首页：`21`
   - 滋补商城：`22/33`
   - 食疗方案：`34`

6. **情感内容中心**
   - 私密男友：`25`
   - 小说漫剧：`26/41`
   - 聊天组合页：`00/00.jpg`

7. **分销中心**
   - 邀请海报：`23`
   - 佣金明细/提现：`24`

8. **我的中心**
   - 我的主页：`40`
   - 关联跳转：预约记录、服务评价、收藏、资产

## 3) 核心逻辑关系（建议）

### 链路A：美美看脸 -> 交易闭环
1) 看脸检测 -> 2) 报告 -> 3) 医生问答 -> 4) 方案 -> 5) 套餐 -> 6) 时间 -> 7) 订单 -> 8) 支付 -> 9) 预约成功/核销码

### 链路B：中医心理问诊闭环
1) 机构选择 -> 2) 医师详情 -> 3) 开始问诊 -> 4) 体质报告 -> 5) 调理方案 -> 6) 到店预约 -> 7) 复诊提醒 -> 8) 服务评价

### 链路C：服务消费闭环
1) 线上/线下瑜伽或上门服务 -> 2) 预约时段 -> 3) 支付 -> 4) 服务完成 -> 5) 评价

### 链路D：内容与分销闭环
1) 内容消费（小说/漫剧/私密男友） -> 2) 邀请闺蜜 -> 3) 佣金累计 -> 4) 提现
